# 증분 OpenAPI 목록과 신청 링크

이 문서는 data.go.kr 공지 `NOTICE_0000000004566`의 OpenAPI 활용신청 링크와 붙임1의 변동분 조회 규칙을 기준으로 생성했습니다.

## 신청 방법

1. 아래 목록에서 필요한 업종의 `신청` 링크를 엽니다.
2. 공공데이터포털에 로그인한 뒤 `활용신청`을 진행합니다.
3. 승인 후 발급된 디코딩 서비스키를 `DATA_GO_KR_SERVICE_KEY` 또는 `MoisClient`에 전달합니다.
4. 여러 업종을 호출해야 하면 각 업종 OpenAPI의 활용 권한을 확인합니다.

## 증분 조회 규칙

- 기본 증분 조건은 `cond[DAT_UPDT_PNT::GTE]=YYYYMMDDHHMMSS`입니다.
- `DAT_UPDT_PNT`는 원천데이터 수정과 개방데이터 보강 수정 시점을 함께 반영합니다.
- 원천데이터 최종수정시점만 기준으로 삼을 때는 `source_modified=True`를 사용해 `LAST_MDFCN_PNT::GTE`로 조회합니다.
- `numOfRows`는 최대 100이며, 전체 동기화는 `totalCount`와 `pageNo` 기반으로 페이지를 넘깁니다.

```python
from mois import MoisClient, list_incremental_openapi_endpoints

client = MoisClient.from_env()

changed = client.get_updated_hospitals("20260505000000")
source_changed = client.get_updated_hospitals(
    "20260505000000",
    source_modified=True,
)

for api in list_incremental_openapi_endpoints():
    print(api.service_slug, api.application_url, api.get_method)
```
